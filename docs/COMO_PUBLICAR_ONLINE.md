# Como publicar online

O Supabase cuida do banco, login e segurança. O site em si deve ficar em uma hospedagem estática.

## Opção mais simples: Netlify

1. Configure `src/config.js`.
2. Entre no Netlify.
3. Arraste a pasta `top-planejados-v60-online` para publicar.
4. Abra o link gerado.

## Opção gratuita com GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos da pasta.
3. Vá em Settings > Pages.
4. Escolha a branch principal e a pasta raiz.
5. Salve e abra o link.

## Opção profissional

Use Vercel ou Cloudflare Pages. Como é HTML puro, não precisa comando de build.

## Comando de build

Não tem.

## Pasta de publicação

Publique a pasta inteira.
