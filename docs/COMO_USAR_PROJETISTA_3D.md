# Como usar o Projetista 3D

O Projetista 3D é um desenho técnico simples para móveis planejados. Ele não é a renderização realista. A renderização fica em outra aba, pronta para API.

## Fluxo recomendado

1. Cadastre um cliente.
2. Crie um projeto, por exemplo: **Cozinha Davi**.
3. Abra a aba **Projetista 3D**.
4. Selecione o projeto.
5. Configure largura, altura e profundidade do ambiente em mm.
6. Arraste móveis da biblioteca ou clique em **Montar cozinha exemplo**.
7. Clique em um móvel para editar medidas, portas, gavetas, prateleiras, material externo e interno.
8. Use 2D para conferência técnica.
9. Use 3D simples para visualizar profundidade.
10. Clique em **Gerar lista de peças**.
11. Clique em **Gerar orçamento deste projeto** para mandar os módulos para orçamento.

## Observação

Esta aba cria um projeto técnico/visual básico, parecido na ideia com Promob: módulos, medidas, portas, gavetas e peças. Ela não gera imagem realista. A imagem realista será feita pela aba Render API quando você conectar a API.
